# Sending emails using Python

1. Register a free account at [https://resend.com](https://resend.com)
2. Get an API key
3. `pip install resend`
4. `pip install python-dotenv`. If in a FastAPI project, you can also use `pip install pydantic-settings`
5. Use code to send emails. Note that, the email may be categorized as SPAM based on your mailbox engine and the email From/Subject/Body.
